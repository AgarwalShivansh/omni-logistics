# Omni-Logistics Predictor

This project predicts cargo shipment delays using a combination of NLP, Deep Learning, and Core Machine Learning.

## Setup Instructions

1. Extract the zip file and open the folder in your terminal/VS Code.
2. Create a virtual environment: `python -m venv venv`
3. Activate the virtual environment:
   - Windows: `.\venv\Scripts\activate`
   - Mac/Linux: `source venv/bin/activate`
4. Install the dependencies: `pip install -r requirements.txt`
5. Run the application: `streamlit run app.py`
